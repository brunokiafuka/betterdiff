## Interactions & Animations

### Hover States
- File list items: Background lightens (#2a2d2e)
- Buttons: Background lightens slightly
- Links: Underline appears

### Click Feedback
- Buttons: 100ms press animation (scale 0.98)
- List items: Instant selection (blue left border)

### Loading States
- Spinner: Simple rotating circle
- Skeleton: Pulsing gray blocks for list items
- Progress: Blue bar for large operations

### Transitions
- Panel switches: 150ms fade
- Modal overlays: 200ms fade + scale
- Hover effects: 100ms ease-out

## Accessibility

- All interactive elements are keyboard accessible
- Tab order follows visual flow
- Focus indicators: 2px blue outline
- Screen reader labels on icons
- ARIA roles for semantic structure

## Responsive Behavior

Minimum window size: 1200Ã—800px

If window shrinks:
1. Right panel becomes collapsible
2. File list becomes collapsible
3. Diff switches to inline mode
4. Status bar shows at bottom

This visual design ensures the app feels like a **professional development tool**, not a web UI crammed into a window.
